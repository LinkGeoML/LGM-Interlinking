LGMInterlinking features¶

class src.featuresConstruction.Features[source]¶

This class loads data and builds the features to be used as input to various classification methods: basic, basic_sorted or lgm.

build_features()[source]¶

Build features and return its values in ndarray of floats.

Returns:
  • fX (ndarray) – The computed features that will be used as input to ML classifiers.
  • y (ndarray) – Binary labels {True, False} to train the classifiers.
